Warroom

Warroom is a private, uncensored community messaging platform built for Andrew Tate — a space where a creator's members talk freely in groups and direct messages, with full admin control over who's in the room.

Overview

Warroom is a private, uncensored messaging platform built for Andrew Tate's members-only community. Rather than relying on mainstream platforms that could moderate or remove content, the client wanted a space they fully control, where their audience can speak freely. Members chat in shared groups and one-to-one private conversations with a full set of modern messaging features — reactions, stickers, replies, voice notes, link previews, and file and video sharing — while real-time touches like typing indicators and delivery receipts keep conversations feeling live. Admins retain complete control, creating groups and inviting, removing, or banning members as needed.

Project Goals

The goal was to give the client a self-owned, censorship-resistant home for their community — a fast, reliable web app for real-time messaging that they, not a third-party platform, ultimately control. It needed instant message delivery and synchronisation, a rich chat experience on par with mainstream messengers, and the moderation tools an admin needs to keep a large, active community in order.

Tech Stack

To achieve this, the following technologies were used:

  • Vue.js:Developed a dynamic and interactive user interface.
  • Tailwind & Vuetify:Applied style for responsive design.
  • Postgres:Utilized a reliable relational database system.
  • Laravel:Implemented a robust backend framework.
  • AWS EC2:Deployed scalable virtual servers.
  • AWS S3:Provided secure object storage.

Features

Reaction

Members can react to any message with emoji, tapping a quick reaction bar to add a thumbs-up, heart, laugh, and more. Reactions sit neatly beneath the message, giving people a lightweight way to respond without cluttering the conversation.

Stickers

A built-in sticker pack — including custom, on-brand designs — lets members express themselves beyond plain text. Stickers and emoji are picked from the same tray in the composer and render large in the chat for extra emphasis.

Reply

Any message can be quoted and replied to directly, so threads stay easy to follow even in a busy group. The original message is shown above the reply, making it clear exactly what someone is responding to.

Sharing Links with Preview

When a member shares a URL, Warroom automatically fetches a rich preview — the page title, description, and image — and renders it as a tidy card. Links become scannable at a glance instead of raw text.

Voice Notes

Members can record and send voice notes straight from the composer, and play them back inline with scrubbing and volume controls. It's a fast, natural way to communicate when typing isn't convenient.

Sharing Files & Videos

Beyond text, members can share files and videos of all kinds. Documents appear as downloadable chips, while videos play inline with full playback controls and a fullscreen option — all backed by AWS S3 storage.

Admin Portal

A separate admin portal gives the client full command of the community. Alongside user management and group management — creating groups and inviting, removing, or banning members — it surfaces analytics such as daily and monthly active users, top-performing groups, and a breakdown of where members are based.

Challenges

Warroom's biggest challenge was delivering real-time, seamless messaging without latency — especially when handling large files, voice notes, and video. On top of that, building a platform the client fully owns and controls meant getting the moderation model right: admins needed reliable tools to create groups and invite, remove, or ban members, so a large and active community could stay well-managed.

Initiatives

To address these challenges, Warroom takes several key initiatives. The platform leverages advanced WebSocket technology for instant message delivery and real-time collaboration. Optimized media handling and compression techniques are implemented to support high-speed file sharing without compromising quality.
